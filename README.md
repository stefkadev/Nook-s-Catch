# Nook's Catch

## Projektbeschreibung
**Nook's Catch** ist ein Arcade-Spiel, das mit Swift und SpriteKit entwickelt wurde. Der Spieler übernimmt die Rolle eines Naturforschers, der herabfallende Gegenstände sammelt. (QUELLE)

## Features
* **Physik-basierte Spielmechanik:** Nutzung von Gravitation und Kollisionen. (QUELLE)
* **Steuerung:** Intuitive Handhabung durch horizontale Wischgesten. (QUELLE)
* **Highscore-System:** Lokale Speicherung von Bestleistungen zur Motivationssteigerung. (QUELLE)
* **Easter Eggs:** Dezente Crossover-Inhalte für Entdecker. (QUELLE)

## Installation
1. Repository klonen oder als ZIP herunterladen.
2. Die Datei `Nook's Catch.xcodeproj` in Xcode öffnen.
3. Ein Zielgerät (z. B. iPhone 15 Simulator) auswählen.
4. Auf den **Run-Button** (Play-Symbol) klicken oder `Cmd + R` drücken.
